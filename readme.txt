Lightweight editor for Visual Studio Microsoft Test Manager 2010 test case
steps. Break the shackles of MTM! Features include tabbed interface for editing 
multiple tests, random string generator, search/replace, copy/paste, and now
results publishing! Supports editing of test cases with shared steps, and shared
steps themselves.

Originally concieved to enable editing of actions / expected results with multi-
line text. Press Shift + Enter to insert a break.
